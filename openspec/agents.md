# Agents 规范文档

## 角色与职责
当前项目由 AI 助手（Sut）进行维护和开发。

## 核心准则
1. **语言**: 保持中文对话。
2. **代码规范**: 
   - 自动修复语法错误、标签闭合问题。
   - 文件头部必须包含版本号注释。
   - 页面容器添加语义化 `id`。
3. **国际化**: 维护 `en`, `zh-CN`, `zh-TW`, `es`, `ar`, `fr`, `pt-BR`, `de`, `ja`, `ko`, `ru` 11 种语言的 JSON 文件。
4. **版本控制**: 遵循 SemVer 2.0.0 规范，更新 `package.json` 和 `changelog.md`。
5. **鲁棒性**: 确保应用包含错误边界 (Error Boundary) 和 404 页面处理。
