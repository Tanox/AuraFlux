# OpenSpec: UI/UX 与交互规范 (05)

## 1. 布局系统
- **Bento Grid:** 设置界面采用模块化 Bento 布局。
- **BottomBar 收窄:** 底部快速控制栏宽度优化为 `max-w-lg`。

## 2. 交互语义 (v1.8.91+)
- **显示选项图标:** 使用 `AdjustmentsHorizontal` 样式。
- **隐藏选项图标:** 使用 `Minus` 样式。
- **全局拖拽:** 优先级 z-index [300] 的音频导入引导层。

## 3. 开关组件规范
- **物理结构:** 开关按钮 (Switch) 必须位于左侧，标签文字 (Label) 位于右侧。
