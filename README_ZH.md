# Aura Flux 🎵👁️

### AI 驱动的 3D 音乐可视化与通感分析引擎 (v1.8.82)

[在线演示](https://aura.tanox.net) | [English](./README.md)

Aura Flux 是一款专业级的 Web 应用程序，可将音频频率转化为高保真生成艺术。基于 **React 18**、**Three.js** 和 **Google Gemini 3 AI** 构建，实现了极致性能与人工智能的完美融合。

## 🚀 核心特性 (v1.8.82)

- **混合渲染管线：** 在高保真 WebGL 3D 场景 (R3F) 与经过优化的离屏 2D Canvas 渲染器之间无缝切换。
- **AI 通感 HUD：** 由 Gemini 3.0 Flash 驱动的实时歌曲识别、情绪分析和动态歌词显示。
- **4K 工作室录制：** 支持以高分辨率（最高 4K）导出您的视觉创作，并支持自定义码率和混音。
- **精制 VJ 控制：** 模块化 Bento 布局，支持双击播放与悬停删除交互，优化后的收窄型底部控制栏。
- **本地隐私优先：** 所有频谱分析和图像渲染均在您的设备本地完成。
