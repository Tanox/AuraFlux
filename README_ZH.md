# Aura Flux 🎵👁️

### AI 驱动的 3D 音乐可视化与通感分析引擎 (v1.8.100)

[在线演示](https://github.com/sutchan) | [English](./README.md)

**Aura Flux** 是一款专业级的 Web 应用程序，可将音频频率转化为高保真生成艺术。基于 **React 18**、**Three.js (WebGL)** 和 **Google Gemini 3 AI** 构建，实现了极致的边缘计算性能与云端智能的完美融合。

---

## 🚀 核心特性

- **混合渲染管线：** 在高保真 WebGL 3D 场景（基于 R3F）与针对低功耗设备优化的 2D Canvas 渲染器之间无缝切换。
- **AI 通感 HUD：** 由 **Gemini 3.0 Flash** 驱动的实时歌曲识别、情绪分析和全量歌词检索。
- **AI 视觉导演：** 让 AI 分析 15 秒音乐，自动为您调配最契合旋律的色彩、速度和视觉模式。
- **4K 工作室录制：** 支持以最高 4K 分辨率导出您的创作，并支持自定义录音增益和码率控制。
- **数据可移植性：** 支持将精密调校的设置导出为 JSON 文件，并在社区中分享您的预设。

## 🌟 使用场景

### 1. 社交媒体创作者 📹
*   **目标：** 为 TikTok、Reels 或 YouTube 音乐视频创建震撼的背景视觉效果。
*   **工作流：** 导入高品质 `.mp3` 或 `.wav` 文件，激活“工作室模式”，将画面比例设为 `9:16`，并录制一段同步表演。
*   **专家建议：** 使用“AI 灵感生成背景”功能，创造一个与歌曲意境完全匹配的独特艺术层。

### 2. 现场 DJ / 主播氛围渲染 🎙️
*   **目标：** 为您的现场 DJ 表演或播客直播添加动态交互视觉。
*   **工作流：** 将音频输入设为“麦克风”（或使用 VB-Audio 等虚拟跳线）。开启“自动隐藏 HUD”和“自动轮播引擎”，无需手动操作即可保持视觉持续进化。
*   **专家建议：** 开启“镜像渲染”以获得极具迷幻感的对称视觉效果。

### 3. 智能家居与沉浸式休息室 🏠
*   **目标：** 将客厅电视转变为一件活生生的数字艺术品。
*   **工作流：** 将 PC/Mac 连接到大屏幕，应用“立体海浪”预设，并开启“时钟图层”。
*   **专家建议：** 使用“流光绸缎”或“神经网络”模式，获得高级、优雅且不干扰注意力的环境体验。

### 4. 禅修、冥想与深度专注 🧘
*   **目标：** 利用节奏感十足的视觉脉动辅助呼吸和专注。
*   **工作流：** 选择“共鸣天体”模式，降低“流动速度”，并使用深蓝色系的调色盘。
*   **专家建议：** 开启“自定义文字”功能，显示您的冥想词或专注关键词。

### 5. 极客与开发者实验室 🧪
*   **目标：** 实验音频频谱密度与信号处理逻辑。
*   **工作流：** 切换到“专业模式”，解锁 FFT 采样大小（最高 2048）、平滑度（惯性）和各通道增益控制。
*   **专家建议：** 开启 FPS 计数器，通过调整“渲染精度”来寻找最适合您 GPU 的平衡点。

## ⌨️ VJ 控制中心 (快捷键)

| 按键 | 动作 |
|:---:|:---|
| `空格` | 开关麦克风 / 播放文件 |
| `F` | 切换全屏 |
| `R` | 随机化美学方案 (随机 Flux) |
| `L` | 开关 AI HUD (歌词与情绪) |
| `H` | 隐藏/显示控制面板 |
| `G` | 开关辉光强度 (Glow) |
| `T` | 开关运动拖尾 (Trails) |
| `← / →` | 切换视觉渲染引擎 |
| `Shift + ↑/↓`| 调节流动速度 |
| `1 - 6` | 在设置标签页之间快速切换 |
| `?` | 显示帮助遮罩层 |

## 🛠️ 技术栈

- **框架：** React 18，使用 Context API 进行全局状态管理。
- **3D 引擎：** Three.js / @react-three/fiber，包含自定义 GLSL 着色器。
- **AI 集成：** 使用 Google Generative AI SDK 调用 Gemini 3.0。
- **音频处理：** 基于 Web Audio API 的快速傅里叶变换 (FFT) 分析。
- **持久化：** IndexedDB (存储大型音频) + LocalStorage (存储配置)。
