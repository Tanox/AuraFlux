# OpenSpec: 安全策略与漏洞报告 (11)

## 1. 隐私保护声明 (v2.1.1)
- **本地优先:** 核心音频分析完全在浏览器沙箱内运行。
- **API 安全:** API 密钥仅在用户本地持有，绝不进行云端上传或中转。
- **匿名请求:** AI 歌曲识别仅发送音频流片段，不携带任何个人元数据。
