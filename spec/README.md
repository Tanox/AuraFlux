## 项目描述
Aura Flux 是一个音乐可视化应用，结合了先进的音频分析、视觉渲染和 AI 集成技术，为用户提供沉浸式的音乐可视化体验。

## 规范标准

### 1. 文档格式
- 所有文档使用 Markdown 格式
- 统一使用 UTF-8 编码，CRLF 行尾
- 文件名使用 `XX_描述.md` 格式，其中 XX 为两位数字序号
### 2. 版本管理
- 采用语义化版本(SemVer): MAJOR.MINOR.PATCH
- 所有规范文档版本号保持同步
- 每次修改递增 PATCH 版本号
### 3. 内容要求
- 清晰的层级结构
- 详细的技术规范
- 统一的术语定义
- 可操作的实现指南
- 代码示例和流程图

### 4. 文档管理
- 定期检查文档完整性
- 确保文档与代码实现一致
- 收集并响应文档反馈
## 维护说明
- 规范文档应与代码实现保持同步
- 新增功能时应更新相应的规范文档
- 定期检查并更新规范以反映最新技术栈和最佳实践
- 建立文档变更记录，与版本号同步
